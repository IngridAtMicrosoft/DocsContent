# Streaming media how-tos and tutorials

## Learn how to stream media with Azure Media Services

Media Services provides extensive guidance for streaming media files. You can stream media by creating a live stream, a video-on-demand (VOD) application or by using the output files of a live stream for a VOD application. Before mezzanine (source) files or media can be streamed, they have to be encoded.

Review the following documents to get an understanding of encoding and live streaming:

* [Encoding](https://docs.microsoft.com/azure/media-services/latest/encode-concept)<br>
* [Live streaming](https://docs.microsoft.com/azure/media-services/latest/media-services-overview)<br>
